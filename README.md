# My Dot-files

## maintenance

maintenance.sh is an arch specific script that does a full update for pacman and AUR helpers, gets rid of cached repositorys, cleans up journal logs, and checks failed services.
its recommended to do system maintenance weekly for arch linux since updates are always coming out. This script made things a little easier and automated

## tmux

my leader key is CTRL+SPACE.

Indexing starts at 1. The only plugin i use is tmux-ressurect so i can pop into sessions on reboot
